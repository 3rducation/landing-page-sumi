# 02 Service

## Service Name
AI自動採点「澄」

## One-line Description
生成AIを活用し、採点・集計・返却業務を効率化する学校向けAI自動採点ツール。

## Short Description
「澄」は、手採点や従来型のOCR中心の採点システムでは対応しづらかった多様な解答パターンに対して、生成AIの文脈理解・意味理解を活用し、採点業務の大幅な効率化を支援するサービスです。

## Core Value Proposition
- 採点時間の大幅削減
- 教員の負担・ストレス軽減
- 採点基準の統一化
- 答案データの集計・返却業務の効率化
- 生徒への迅速なフィードバック
- 教職員が本来注力すべき教育活動へ時間を使える状態をつくる

## Key Claims
- 採点時間を約75%削減
- 生成AIを活用した柔軟な採点
- 採点基準の一貫性向上
- 集計・返却業務の効率化

## Claim Notes
- 「約75%削減」は提案資料内の記載に基づく。LP上では必要に応じて「弊社調査」または「社内検証」と補足する。
- 断定しすぎず、「削減を目指す」「削減が期待できます」などの表現も検討する。

## Problems Solved
- 手採点に時間がかかる
- 従来型の採点システムでも、問題数自体は減らない
- 表記揺れや同義表現に対応しづらい
- 記述問題や図への書き込み問題に対応しづらい
- 採点後の集計・返却に時間がかかる
- 採点基準が担当者ごとにぶれやすい

## Examples of Flexible Grading
- 1 vs ①
- あ vs ア
- 2(x-1) vs 2x-2
- 二酸化炭素 vs CO2
- 記述問題
- 図に書き込む問題

## Features
1. 生成AIを活用した採点支援
2. 答案データの集計
3. 採点結果の返却支援
4. 採点基準の統一化
5. クラス平均点・偏差値・得点分布の確認
6. 採点修正後の再送
7. 生徒への一括返却

## Do Not Say
- 教員を置き換える
- 完全自動で絶対に正しい
- 100%ミスがない
- 個人情報管理が完璧
- すべての問題形式に無条件対応

## Preferred Expressions
- 教員の採点業務を支援
- 先生が確認・修正できる
- 採点業務を効率化
- 採点基準の統一化を支援
- 生徒への迅速なフィードバックを支援
- 学校現場に寄り添う
