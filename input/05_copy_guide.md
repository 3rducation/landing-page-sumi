AI自動採点「澄」は、採点時間の削減、採点基準の統一化、生徒への迅速なフィードバックを支援します。

## Problem Copy
採点業務は、単に答案を採点するだけでは終わりません。問題ごとの確認、採点基準の調整、集計、返却、生徒へのフィードバックまで、多くの時間と集中力が必要です。

## Solution Copy
AI自動採点「澄」は、生成AIを活用して採点業務を支援し、集計・返却までの流れを効率化します。先生が最終確認・修正できる運用を前提に、学校現場に寄り添った採点DXを実現します。

## Feature Short Copy
- 生成AI採点支援: 表記揺れや多様な解答パターンを考慮した採点支援
- 採点基準の統一化: 一貫した基準に基づく採点を支援
- 集計の効率化: 答案データを集計し、クラス全体の傾向を把握
- 返却の効率化: 生徒へのフィードバック返却を効率化
- 分析: クラス平均点、偏差値、得点分布を確認
- 確認・修正: 先生がAIの採点結果を確認・修正可能
